Essentials:

  • This template is based on Bootstrap v3.3.6 responsive framework
  • Chrome for Win or Mac (use the inspector, right click -> inspect element to make your life easier)
  • A decent text editor like Atom for Win or Text Wrangler for Mac
  • When using Variant - Run in Chrome or Firefox for full features and best performance (Safari is unable to export files and IE is not supported at this time)

1. Countdown Timer

The countdown timer is controlled by a javascript plugin called 'The Final Countdown'. We have set it up, so that all you need to do is to pass a date into the data attribute 'data-date' in the format YYYY/MM/DD.

2. Using the Icon Pack

Multi-concept layout make use of two icon pack one is ionicons pack and second is Themify Icons. Using Ionicons icon packs is a simple matter of knowing the correct class to use.

4.Mailchimp Subscribe Form

Verifing mailchimp subscribe form is very easy in Deft,

1. Replace the form action url with your own from the MailChimp supplied embed code
2. Within the action url, replace "subscribe/post" with "subscribe/post-json"
3. Be sure the action url starts with http:// If it doesn't, the form will hang when testing the html as a local file in your browser.

6. Background Color settings for Angle Animation in Restaurant page

To change those colors go to js folder and edit angle.js file with any text editor. then go to line 1604 and 1605 you will find some code like this

ambient: '#09ffc5',
diffuse: '#048A81',

Just replace the hex values with your color hex values and remember ambient color is for background and diffuse is the little portion which move when you hover. and only use hex values.
